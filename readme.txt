Prevent Wordpress from adding noreferrer and noopener to external links.

== Description ==

No noopener noreferrer plugin will remove rel="noopener" and rel="noreferrer" from the external links


== Installation ==

1. Download the plugin.
2. Upload to your blog (/wp-content/plugins/).
3. Activate it.

You're done!

Uninstalling is as simple as deactivating the plugin.
